# ベイズ最適化デモ

## 環境構築

ryeがインストールされていることを前提とします。
ref. [Installation - Rye](https://rye-up.com/guide/installation/)

以下のコマンドでライブラリをインストールします。

```sh
rye sync
```

OSXの場合は次の手順を踏んでAxをインストールすることでアプリの高速化が可能です: [Installation - Ax](https://ax.dev/docs/installation.html)

以下のコマンドでサーバーを起動します。

```sh
rye run start
```
